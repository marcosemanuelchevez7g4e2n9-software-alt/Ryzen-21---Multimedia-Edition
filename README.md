# Ryzen-21---Multimedia-Edition
Proyecto de aplicación multiplataforma que se encargará de brindar servicio para registro de asistencia académica y institucional a nivel nacional e internacional, con funciones multimedia integradas en su interfaz y contenido.

Este repositorio será el almacen central de administración y manejo de datos, archivos y código relacionado al desarrollo y programación del proyecto durante todas y cada una de las etapas de desarrollo, todo esto con el objetivo de guardar y recuperar archivos importantes (como fragmentos de código, frameworks, librerías y componentes, notas de codificación, listas de herramientas de software, cronogramas de actividades en equipo y individuales, procesos lógicos y estructurales) para ser utilizados como referencia o recurso durante el desarrollo y creación del proyecto, algo vital al trabajar en proyectos de software o sistemas informáticos.

También, se podrán almacenar bases de datos, recursos de programación, prórroga de lenguajes de programación, partes de programación del proyecto, créditos de diseño, notas del equipo, aspectos y observaciones técnicas, lingüística técnica, criterios de progreso en el proyecto, menciones honoríficas, reglas de distribución y desarrollo, etc. Todo esto como algo fundamental para trabajar durante el ciclo de vida del proyecto con el equipo y de manera individual.
